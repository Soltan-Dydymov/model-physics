# 4.9. Движение небесного тела в гравитационном поле 

## Основные законы

**Второй закон Ньютона:**
$$\vec{F} = m \vec{a}$$

**Закон всемирного тяготения:**
$$F = G \frac{Mm}{r^2}$$

**В векторной форме:**
$$\vec{F} = -G \frac{Mm}{r^3} \vec{r}$$

где $G = 6{,}67 \times 10^{-11}$ Н·м²/кг² — гравитационная постоянная, r — расстояние между центрами тел

## Система дифференциальных уравнений

$$\frac{dv_x}{dt} = -GM \frac{x}{(x^2+y^2)^{3/2}}$$

$$\frac{dv_y}{dt} = -GM \frac{y}{(x^2+y^2)^{3/2}}$$

$$\frac{dx}{dt} = v_x$$

$$\frac{dy}{dt} = v_y$$

## Начальные условия

$$x(0) = x_0, \quad y(0) = 0$$

$$v_{x0} = v_0 \cos \alpha, \quad v_{y0} = v_0 \sin \alpha$$

## Условие замкнутости траектории

Характер орбиты определяется полной механической энергией системы:

$$E = \frac{v^2}{2} - \frac{GM}{r}$$

**Критерий замкнутости:**

| Условие | Тип орбиты | Траектория |
|---------|------------|------------|
| E < 0 | Эллипс | Замкнутая, тело остаётся связанным |
| E = 0 | Парабола | Предельный случай покидания гравитационного поля |
| E > 0 | Гипербола | Незамкнутая, тело покидает систему |

**Скорость убегания:**

На расстоянии r от Солнца:

$$v_{esc} = \sqrt{\frac{2GM}{r}}$$

Для r₀ = 100 а.е.: v_esc = 4.21 км/с

v₀ = 2 км/с < 4.21 км/с, кинетической энергии кометы недостаточно для преодоления гравитационного притяжения Солнца, поэтому траектория замкнутая

## Проверка законов Кеплера

**Первый закон Кеплера**

> **Каждая планета движется по эллипсу, в одном из фокусов которого находится Солнце**

Проверка для тестовой орбиты v₀ = 1.5 км/с:
- Сумма r_min + r_max должна равняться 2a (свойство эллипса)
- Перигелий: числ. = 3.2735 а.е., теор. = 3.2735 а.е.
- Афелий: числ. = 111.3002 а.е., теор. = 111.3001 а.е.
- Сумма r_min + r_max = 114.5737 а.е., теоретически 2a = 114.5736 а.е.
- Ошибка: < 0.001%

**Подтвержден, траектория является эллипсом с Солнцем в фокусе**

---

### Второй закон Кеплера

> **Радиус-вектор планеты в равные времена описывает равные площади**

Математически это означает постоянство секториальной скорости:

$$\frac{dA}{dt} = \frac{1}{2}|x \cdot v_y - y \cdot v_x| = \frac{L}{2m} = const$$

Этот закон является следствием сохранения момента импульса L

Проверка:
- Средняя секториальная скорость: 5.625×10¹⁵ м²/с
- Стандартное отклонение: 1.85×10⁷ м²/с
- Относительное отклонение: < 0.001%

**Подтвержден, секториальная скорость постоянна с высокой точностью**

---
